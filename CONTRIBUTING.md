# Contributing to DarkTorch

We welcome contributions to DarkTorch!

## How to Contribute

1. **Fork the repository**.
2. **Create a branch** for your feature or fix (`git checkout -b feature/amazing-feature`).
3. **Commit your changes** (`git commit -m 'feat: Add amazing feature'`).
4. **Push to the branch** (`git push origin feature/amazing-feature`).
5. **Open a Pull Request**.

## Development Setup

1. Ensure Docker and Docker Compose are installed.
2. Run `docker-compose up --build` to start the development environment.
3. Frontend is at `http://localhost:80`, Backend at `http://localhost:5000`.

## Code Style

- **Backend**: Python (FastAPI). Follow PEP 8.
- **Frontend**: React (TypeScript). Keep components clean and functional.
